# Breakout

## Usage
* **Train DQN:** 
    -`$ python main.py --test_dqn`  with 10000 episodes
    -`$ python main.py --test_dqn N`  with N episodes
* **Test Agent:**
    -`$python main.py --train_dqn` train without render
    -`$ python main.py --train_dqn --render` train with render
